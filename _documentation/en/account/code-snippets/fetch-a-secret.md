# Fetch a secret

This will return the details of a single secret. The value of the secret will
never be shown.

```snippet_variables
- VONAGE_API_KEY
- VONAGE_API_SECRET
- ACCOUNT_ID
- VONAGE_SECRET_ID.FETCH
```
